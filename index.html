Contacts

▶

Capsule holds all your contacts — clients, prospects, networking connections, referral partners, suppliers.

Import status

Prospects and clients have been imported from Monday emails. Tags are applied to distinguish prospects from clients.

Organisation fields

Each organisation record can include industry (from gov.uk categories) and organisation size.

Contact relationships

Contact relationship types have been defined — main contact, finance contact, and PA. These help you quickly identify who to speak to at each organisation.

When adding a new contact, always tag them as Prospect or Client. Tags drive mailing list membership in Transpond.

Sales pipeline

▶

The sales pipeline replaces the Genie AI pipeline. The stages mirror your current workflow:

StageWhat it means
New EnquiryAn enquiry has come in — referred, networking, website booking
Unable to ContactTried to reach them, no response (try 3 times, then park for 3-6 months)
Discovery Call BookedJulia has a call scheduled
Proposal SentKelly has sent a proposal via Better Proposals
Follow UpWaiting for a response, chasing the decision
WonThey've said yes — create a project
LostThey've said no or gone quiet
Not ReadyInterested but not ready to commit — park and follow up later

After the discovery call

Julia records a standard Loom welcome video that is sent to the prospect after the discovery call — a personal touch before the proposal arrives.

Follow-up cadence

After sending a proposal: follow up at 3 days, then again at 5 days. If no response after the second follow-up, move to Lost.

Better Proposals integration is basic — it doesn't auto-populate Capsule. When you send a proposal, manually update the opportunity and move it to "Proposal Sent".
4

Projects & client work

▶

When an opportunity is won, it becomes a project in Capsule. This replaces Asana for tracking active client work.

Active Client Work board

The Active Client Work board has been set up in Capsule. All active client projects should be assigned to this board so they appear in the board view.

If a project isn't on the board, it won't appear in the board view — only in the list view. To fix: go to the project → Edit → Board → select "Active Client Work" → choose the correct stage → Save. See Tutorial Video 1 for a walkthrough.

Asana to Capsule mapping

Asana conceptCapsule equivalent
Project (per client)Project on the Active Client Work board
Sections (Employee Lifecycle, HR Advice, etc.)Stages on the board, or separate projects per workstream
Tasks within sectionsTasks on the project
Comments on tasksNotes / Log Activity

Retainer clients

Retainer clients have multiple projects within their contract — HR Advice, Employee Lifecycle, Casework, Recruitment. Each workstream can be a separate project on the board, giving Siobhan visibility across all active work.

Asana migration

Migration is in progress using a batched approach — all retainer clients first, then ad-hoc clients. Each Asana project becomes a Capsule project with tasks.

5

Email integration

▶

Capsule connects to Outlook via the email sidebar.

Email signature

The email signature has been configured in Capsule. The signature image is hosted externally (not embedded in Outlook) — copy-paste the image into Capsule's signature field so it renders consistently across Capsule-sent emails.

SSO issue (resolved)

Microsoft 365 single sign-on was interfering with Capsule login. This has been disabled under Account Settings → Microsoft 365 → Disable. Standard username/password login is now restored.

Starter plan (current)

Manual email storage only. You choose which emails to store. Only outgoing emails are visible. Incoming replies don't appear automatically.

Growth plan (upgrading to)

Automatic email storage per contact. Once enabled, all interactions (incoming and outgoing) are stored automatically.

The Growth upgrade is needed for the email workflow to work properly. Without it, the team has to manually store every email — which won't happen consistently.
6

Transpond (email marketing)

▶

Transpond replaces the marketing email functionality in Genie AI. The Transpond account has been linked to Capsule.

Tags and mailing lists

Tags in Capsule drive mailing list membership in Transpond. When you tag a contact as Prospect or Client, they are automatically added to the corresponding mailing list.

Domain verification

DNS records need to be added via IONOS hosting to verify the peoplepillar.co.uk domain. This is still pending. Without verification, marketing emails are more likely to end up in spam. The records are additive — adding them won't break anything.

Forms

Transpond forms can capture email addresses for paid marketing campaigns (e.g. the new course landing page). Forms can be embedded or shared as a link.
